## üíª How to Run

To run these programs, you'll need a C compiler (like `gcc` or `clang`).

1. **Compile a file:**
   ```bash
   clang filename.c -o output_name
   ```
   *Example:*
   ```bash
   clang guessingGame.c -o game
   ```

2. **Run the executable:**
   ```bash
   ./output_name
   ```
   *Example:*
   ```bash
   ./game
   ```
